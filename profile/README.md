### [@reechy-tools/recorder](https://github.com/reechy-tools/recorder)

OBS-style screen + camera PIP recording in the browser.  
Draggable overlay · Shape masks · Audio mixing · React hooks · ~10KB gzipped.

```bash
npm install @reechy-tools/recorder
```

Built and battle-tested at **[Reechy](https://reechy.cam)** — browser-based video pitch recorder for sales, hiring, and async communication.
